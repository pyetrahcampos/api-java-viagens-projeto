# API Agência de Viagens - Desafio Spring Boot

Esta é uma API RESTful desenvolvida para uma agência de viagens, com o objetivo de permitir que outras empresas de turismo e aplicativos de terceiros integrem-se ao sistema para gerenciar destinos e pacotes de viagens.

O projeto foi desenvolvido como parte de uma atividade avaliativa, focando na correta implementação dos métodos HTTP, boas práticas de programação e separação de conceitos através da camada de serviço (Service).

---

## Tecnologias Utilizadas

* **Java 21**
* **Spring Boot 4.x** 
* **Maven** 
* spring web, lombok
* **SpringDoc / Swagger** 

---

##  Como Executar o Projeto

1. Certifique-se de ter o **Java 21** instalado na sua máquina
2. Importe o projeto pelo eclipse
3. Faça o download das dependências 
4. Execute a classe principal `Desafio2Application.java` 
5. A API iniciará por padrão na porta **8081**
